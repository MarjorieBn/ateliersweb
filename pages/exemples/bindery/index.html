Bindery.js est une bibliothèque javascript pour créer des livres imprimables avec HTML et CSS.

Le contenu est divisé sur plusieurs pages, puis vous configurez des règles pour générer des en-têtes, des double-pages et des notes de bas de page. Transformez les liens en numéros de page pour créer une table des matières ou un index. Lorsque vous êtes prêt·e à imprimer, configurez les bord-perdus, les repères de coupe et générez l’imposition.

Si vous êtes designer web, vous pouvez considérer les livres comme une extension du design responsive. Si vous êtes designer print, vous pouvez produire des mises en page programmatiquement, sans nécessiter InDesign.
